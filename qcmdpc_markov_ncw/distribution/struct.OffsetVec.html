pub struct OffsetVec<T> {
    pub offset: usize,
    pub stride: usize,
    pub vec: Vec<T>,
}
Expand description

A vector with offset and stride for efficient distribution operations.

Fields§

§offset: usize

Offset of the vector.

§stride: usize

Stride between elements, where v[i] = v'[offset + stride * i]

§vec: Vec<T>

The underlying vector data.
