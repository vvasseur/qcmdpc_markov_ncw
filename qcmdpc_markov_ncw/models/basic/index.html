qcmdpc_markov_ncw::models

Module basic

Source
Expand description

This module provides the basis models that other derived transition models can build upon.

The available base models track:

  • Syndrome weight (s) and error weight (t): A simple model
  • STU Specific: Building on (s, t, u) for keys with known Tanner graph near codeword degree distributions.
