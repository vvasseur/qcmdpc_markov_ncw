Expand description

Counters associated with states in MDPC code decoding.

This trait defines methods for manipulating and analyzing counters that are used in the decoding process of MDPC codes. It provides functionality for conditioning counters, computing locking probabilities, applying sampling, and computing transitions.

§Type Parameters

  • T: A tuple type representing the state associated with these counters.
