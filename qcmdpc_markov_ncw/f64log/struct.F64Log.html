pub struct F64Log { /* private fields */ }

A type for performing arithmetic in log domain to handle very large or small numbers without overflow/underflow. Stores values as natural logarithm with a sign.

Compute a single coefficient in the hypergeometric distribution for a population with a given sample size and number of draws.

The formula is:

P(X = k) = binomial(sample_size, k) * binomial(pop_total - sample_size, draws - k) / binomial(pop_total, draws)

pub fn max_n_pmf(pmf: &[F64Log], n: usize) -> Vec<F64Log>

Calculates the probability mass function (PMF) of the largest order statistic in a sample of size n from a set of iid random variables described by a probability mass function (PMF).

Rather than computing differences of CDFs, which can lead to numerical instability, we use the binomial expansion: P(max X_i = k) = sum_{i=1}^n binom(n,i) * P(X<k)^(n-i) * P(X=k)^i

§Arguments
  • pmf - Input probability mass function
  • n - Sample size
§Returns

PMF of maximum value in sample
