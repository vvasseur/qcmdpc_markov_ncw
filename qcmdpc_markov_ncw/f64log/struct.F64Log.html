pub fn max_n_pmf(pmf: &[F64Log], n: usize) -> Vec<F64Log>

Calculates the probability mass function (PMF) of the largest order statistic in a sample of size n from a set of iid random variables described by a probability mass function (PMF).

Rather than computing differences of CDFs, which can lead to numerical instability, we use the binomial expansion: P(max X_i = k) = sum_{i=1}^n binom(n,i) * P(X<k)^(n-i) * P(X=k)^i

ยงArguments
  • pmf - Input probability mass function
  • n - Sample size
